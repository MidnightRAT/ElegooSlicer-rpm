# Plan: Build ElegooSlicer x86_64 RPM for Fedora

## Target
- **Package**: ElegooSlicer v1.5.2.2
- **Source**: https://github.com/ELEGOO-3D/ElegooSlicer/tree/v1.5.2.2
- **Target**: x86_64 RPM for Fedora 44, installed to `/opt/ElegooSlicer`
- **Environment**: Fedora 44 container, GCC 16.1.1, 31GB RAM, 84GB free disk

## Architecture Overview

ElegooSlicer is a fork of OrcaSlicer/PrusaSlicer. The build has two stages:
1. **deps/** - ExternalProject-based build that downloads and compiles all dependencies from source into `deps/build/destdir/usr/local/`
2. **Main build** - Uses `CMAKE_PREFIX_PATH` to find the deps, compiles the app

The deps system downloads from GitHub/internet during build. Most heavy deps (Boost, TBB, wxWidgets, OCCT, OpenCV, CGAL, OpenVDB, etc.) are built from source regardless of system availability.


## Step 3: Build Dependencies

The deps/ CMake system handles all third-party dependencies. We run:

```bash
cd src
mkdir -p deps/build
cmake -S deps -B deps/build -G Ninja \
  -DCMAKE_BUILD_TYPE=Release \
  -DCMAKE_INSTALL_PREFIX=deps/build/destdir/usr/local \
  -DDESTDIR="" \
  -DDEP_WX_GTK3=ON \
  -DSLIC3R_PCH=OFF

cmake --build deps/build -j$(nproc)
```

**Note**: This downloads and builds: Boost 1.83+, TBB, wxWidgets 3.2, OCCT 7.x, OpenCV, CGAL, OpenVDB, GLEW, GLFW, Cereal, NLopt, Sentry Native SDK, ixwebsocket, PahoMqttCpp, elegoolink, and more. This step takes significant time (30-60+ min depending on CPU).

**Dependency on internet**: The deps/ system uses `ExternalProject_Add` with `GIT_REPOSITORY` and `URL` to download sources. Must have internet access during build.

## Step 4: Build ElegooSlicer

```bash
cd src
mkdir -p build
cmake -S . -B build -G "Ninja Multi-Config" \
  -DCMAKE_PREFIX_PATH="$(pwd)/deps/build/destdir/usr/local" \
  -DSLIC3R_STATIC=1 \
  -DSLIC3R_GTK=3 \
  -DSLIC3R_FHS=0 \
  -DBBL_RELEASE_TO_PUBLIC=1 \
  -DBBL_INTERNAL_TESTING=0 \
  -DELEGOO_INTERNAL_TESTING=0 \
  -DSLIC3R_PCH=OFF

cmake --build build --config Release --target ElegooSlicer -j$(nproc)
```

## Step 6: Create Package Layout

The build produces a `build/src/Release/elegoo-slicer` binary. We need to create a package layout for RPM:

```
/opt/ElegooSlicer/
  bin/
    elegoo-slicer          # binary
    libaosl.so             # Agora
    libagora_rtm_sdk.so    # Agora
  resources/               # copied from source tree
  elegoo-slicer.sh         # wrapper script (sets LD_LIBRARY_PATH, etc.)
```

## Step 7: Create RPM Spec File

Create `elegoo-slicer.spec` in the project root. Key sections:

```spec

Version:        1.5.2.2
Release:        1%{?dist}
Summary:        Open-source slicer for FDM 3D printers
License:        AGPL-3.0
URL:            https://github.com/ELEGOO-3D/ElegooSlicer
Source0:        %{name}-%{version}.tar.gz

# Runtime deps - minimal for self-contained build

Requires:       gtk3
Requires:       webkit2gtk4.1
Requires:       mesa-libGL
Requires:       mesa-libEGL
Requires:       dbus-libs
Requires:       libX11
Requires:       pango
Requires:       fontconfig
Requires:       freetype
# Bundled: boost, tbb, wxwidgets, occt, opencv, cgal, opendb, glew, glfw,
#           openssl, curl, png, jpeg, eigen, cereal, nlopt, sentry, etc.

# Build deps
BuildRequires:  cmake >= 3.13
BuildRequires:  ninja-build
BuildRequires:  gcc gcc-c++
BuildRequires:  pkgconf
BuildRequires:  autoconf automake libtool m4
BuildRequires:  git wget
BuildRequires:  dbus-devel gtk3-devel webkit2gtk4.1-devel
BuildRequires:  glew-devel glfw-devel mesa-libGLU-devel mesa-libGL-devel
BuildRequires:  libjpeg-turbo-devel libpng-devel
BuildRequires:  openssl-devel libcurl-devel
BuildRequires:  freetype-devel fontconfig-devel pango-devel
BuildRequires:  eigen3-devel cereal-devel
BuildRequires:  extra-cmake-modules eglexternalplatform-devel
BuildRequires:  gstreamer1-devel gstreamer1-plugins-base-devel gstreamermm-devel
BuildRequires:  wayland-protocols-devel libxkbcommon-devel
BuildRequires:  libX11-devel libXi-devel libXrandr-devel libXinerama-devel
BuildRequires:  libXcursor-devel libXcomposite-devel libXdamage-devel libXext-devel
BuildRequires:  at-spi2-core-devel libepoxy-devel
BuildRequires:  libspnav-devel libsecret-devel libmspack-devel
BuildRequires:  perl-FindBin texinfo
BuildRequires:  file

%description
ElegooSlicer is an open-source slicer compatible with most FDM printers.

%prep
%autosetup -n ElegooSlicer-%{version}

%build
# Build deps first
mkdir -p deps/build
cmake -S deps -B deps/build -G Ninja \
  -DCMAKE_BUILD_TYPE=Release \
  -DDESTDIR="" \
  -DDEP_WX_GTK3=ON \
  -DSLIC3R_PCH=OFF
cmake --build deps/build -j$(nproc)

# Build main app
mkdir -p build
cmake -S . -B build -G "Ninja Multi-Config" \
  -DCMAKE_PREFIX_PATH="$(pwd)/deps/build/destdir/usr/local" \
  -DSLIC3R_STATIC=1 \
  -DSLIC3R_GTK=3 \
  -DBBL_RELEASE_TO_PUBLIC=1 \
  -DBBL_INTERNAL_TESTING=0 \
  -DELEGOO_INTERNAL_TESTING=0 \
  -DSLIC3R_PCH=OFF
cmake --build build --config Release --target ElegooSlicer -j$(nproc)

mkdir -p %{buildroot}/opt/ElegooSlicer/resources

cp build/src/Release/elegoo-slicer %{buildroot}/opt/ElegooSlicer/bin/

# Install Agora libs if they exist
cp -f build/src/Release/libaosl.so %{buildroot}/opt/ElegooSlicer/bin/ 2>/dev/null || true
cp -f build/src/Release/libagora_rtm_sdk.so %{buildroot}/opt/ElegooSlicer/bin/ 2>/dev/null || true

# Install resources
cp -R resources/* %{buildroot}/opt/ElegooSlicer/resources/

# Install wrapper script
cat > %{buildroot}/opt/ElegooSlicer/elegoo-slicer.sh << 'WRAPPER'
#!/bin/bash
DIR=$(dirname "$(readlink -f "$0")")
export LD_LIBRARY_PATH="$DIR/bin:$LD_LIBRARY_PATH"
exec "$DIR/bin/elegoo-slicer" "$@"

# Desktop file
mkdir -p %{buildroot}/usr/share/applications
cat > %{buildroot}/usr/share/applications/elegoo-slicer.desktop << 'DESKTOP'

Name=ElegooSlicer
Comment=Open-source slicer for FDM 3D printers
Exec=/opt/ElegooSlicer/elegoo-slicer.sh %f

Categories=Utility;Engineering;

# Icons
for size in 32 122 192; do
  mkdir -p %{buildroot}/usr/share/icons/hicolor/${size}x${size}/apps
  cp resources/images/ElegooSlicer_${size}px.png \
     %{buildroot}/usr/share/icons/hicolor/${size}x${size}/apps/elegoo-slicer.png 2>/dev/null || true

%files
/opt/ElegooSlicer
/usr/share/applications/elegoo-slicer.desktop
/usr/share/icons/hicolor/*/apps/elegoo-slicer.png

%changelog
```

## Step 8: Build RPM

```bash
# Set up rpm build tree
rpmdev-setuptree  # or mkdir -p ~/rpmbuild/{BUILD,RPMS,SOURCES,SPECS,SRPMS}

# Create tarball
tar czf ~/rpmbuild/SOURCES/elegoo-slicer-1.5.2.2.tar.gz -C src .

# Build RPM
rpmbuild -ba elegoo-slicer.spec
```

## Step 9: Install & Test

```bash
sudo dnf install ~/rpmbuild/RPMS/x86_64/elegoo-slicer-1.5.2.2-1.fc44.x86_64.rpm
```

## Key Files to Create/Modify

1. **`elegoo-slicer.spec`** - RPM spec file (main deliverable)
2. **Build wrapper script** - Optional helper that runs the full build pipeline
3. No source code modifications needed (builds as-is from upstream)

## Important Notes

1. **Build time**: The deps/ build downloads and compiles ~30+ libraries from source. Expect 30-120 minutes.
2. **Internet required**: deps/ uses ExternalProject_Add to download sources from GitHub.
3. **Memory**: Build uses significant memory (recommend 8GB+ per job).
4. **Agora SDK**: The Agora RTM SDK is proprietary; it's downloaded by the elegoolink build step from GitHub. If unavailable, the build may fail or the app may lack chat features.
5. **Sentry**: Built from source via deps/. Provides crash reporting.
6. **wxWidgets 3.2**: System wxGTK 3.2.9 is available but the deps system builds its own. Using system wxWidgets would require patching deps/CMakeLists.txt.
7. **RPM provides**: The RPM will list only runtime library dependencies, not build-time ones.
